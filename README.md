# 🟡 PokéVerse

A modern and responsive Pokémon web application built using **React**, **Vite**, **Tailwind CSS**, and **PokéAPI**.

## 🌐 Live Demo

**https://pokeverse-gules.vercel.app/**

## ✨ Features

* 🔍 Search Pokémon by name
* 📖 Detailed Pokémon information
* ❤️ Add Pokémon to Favorites
* ⚔️ Compare Pokémon stats
* 🌍 Browse different Pokémon regions
* 👑 Legendary Pokémon section
* 🎨 Modern responsive UI
* ⚡ Fast performance with Vite

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript
* PokéAPI

## 🚀 Installation

```bash
git clone https://github.com/nitish19work-ctrl/pokeverse.git

cd pokeverse

npm install

npm run dev
```
